Home Networking Project

A hands-on project demonstrating physical computer networking, IP addressing, and file sharing between two PCs

Project Overview

This project involved creating a direct connection between two computers using Ethernet cables, configuring IP addresses manually, and setting up file sharing through Windows Control Panel and File Explorer options.

Implementation Steps

1

Physical Connection

Connected two computers directly using an Ethernet crossover cable for direct communication without a router or switch.

2

IP Addressing

Went to Control Panel > Network and Sharing Center > Change adapter settings. Assigned static IP addresses from the same subnet to both computers (e.g., 192.168.1.10 and 192.168.1.20 with subnet mask 255.255.255.0).

3

Sharing Configuration

Enabled network discovery and file sharing in Advanced sharing settings. Created shared folders with appropriate permissions.

4

Testing

Verified connection using ping command and tested file transfer between the two computers.
